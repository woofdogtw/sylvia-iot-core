# sylvia-iot-coremgr-cli

The command-line tool for Sylvia-IoT core manager.

This CLI wraps following APIs:
- `sylvia-iot-auth` auth APIs
- `sylvia-iot-coremgr` APIs
- `sylvia-iot-data` APIs
